# Pet

Pet is a small browser-based interactive project built using HTML, CSS, and JavaScript.

The application simulates a virtual pet that reacts to user activity.  
The pet changes its state depending on cursor movement and inactivity.  
For example:
- Fast cursor movement makes the pet hungry
- Normal movement keeps the pet happy
- No movement for a few seconds makes the pet sleepy

This project demonstrates basic concepts such as event handling, animations, and simple state management in JavaScript.
